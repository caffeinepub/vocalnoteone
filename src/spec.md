# Specification

## Summary
**Goal:** Transform the three action controls into prominent rectangular blue buttons with icons and French text labels.

**Planned changes:**
- Create three distinct rectangular buttons with solid blue backgrounds at the bottom of the screen
- Add microphone icon with "Enregistrer mes notes" text to the first button
- Add paper/document icon with "Voir mes notes" text to the second button
- Add trash/bin icon with "Effacer mes notes" text to the third button
- Display all text labels in black or gray color on the blue backgrounds

**User-visible outcome:** Users will see three clearly visible, clickable blue rectangular buttons at the bottom of the screen, each displaying an icon and French label for recording, viewing, and deleting notes.
